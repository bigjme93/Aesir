Welcome to Aesir.

Aesir is an attempt to step unRAID up a gear.

unRAID is a fantastic Home media NAS solution, but the interface doesn't reflect the quality of the underlying product, as unRAID enthusiasts we decided to take on the project of making an awesome GUI that would showcase the quality of unRAID.

Technologies used

  • SASS
  • Codeigniter 3.0.0 ( With additional Modular Extensions HMVC )
  • PHP 5.4+
  • jQuery
  • SQLite3
